## ğŸ”„ Spaced Repetition Methodology

This repository implements spaced repetition to move solutions from short-term to long-term memory:

1. **First Run**: Solve the problem for the first time, understanding the approach
2. **Second Run**: Revisit after a few days/weeks, solve without looking at previous solution
3. **Third Run**: Solve again after a longer interval to reinforce the pattern
4. **Fourth+ Runs**: Continue until the solution becomes intuitive
